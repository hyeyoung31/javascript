Loop

for 반복문

        for(초기문;조건문;증감){
            반복할 코드
        }

while 반복문

        초기문;
        while(조건문){
            //반복
            증감
        }

do while 반복문

        초기문;
        do{
            //반복
            //증감
        }while(조건);

forEach 반복문

for in 반복문

        in 뒤에 객체명을 입력하고 객체 안의 각각의 속성(property)을 item 으로 지정하여
        출력하고 객체에서 각 속성의 값은 객체명[속성명]의 문법으로 출력

for of 반복문

IE에서는 작동하지 않는 구문. 배열의 값을 letter(item) 변수로 할당받아 해당 값을 출력함
