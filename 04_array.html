Array

        배열은 비슷한 성격의 여러 데이터들을 한 저장소에 저장해 놓고 
        필요할 때 데이터를 꺼내서 사용할 때 생성한다.

        새배열

        
        var 이름 = new Array();
        var 이름 = [];
        
        배열명[0] = 값;

        값 확인하기
        배열명[1]

        배열의 개수 배열명.length

join

      배열의 값을 하나의 문자열로 합쳐준다(join);
      배열명.join('구분자');

reverse

        원소의 정렬를 역순 변경(원본 수정!)
        배열명.reverse();

sort

        원소들을 오른차순, 내림순 정렬 (원본 수정)
        배열명.sort();

push

기존 배열에 마지막에 값을 추가

pop

기존 배열에 마지막 값을 제거

unshift

배열의 처음에다 값을 추가

shift

배열의 처음에다 추가 된 값 제거

slice

배열에서 원하는 부분 복사.원본보존

splice

배열에서 원하는 부분 잘라내기. 원본바뀜

concat

        원본.concat(새로운배열)

filter

        조건에 부합하는 원소들을 추출하여 새로운 배열을 생성
