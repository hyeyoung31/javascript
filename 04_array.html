Array

        배열은 비슷한 성격의 여러 데이터들을 한 저장소에 저장해 놓고 
        필요할 때 데이터를 꺼내서 사용할 때 생성한다.

        새배열

        
        var 이름 = new Array();
        var 이름 = [];
        
        배열명[0] = 값;

        값 확인하기
        배열명[1]

        배열의 개수 배열명.length

join

      배열의 원소들을 연결하여 하나의 값(문자)로 만들 수 있다.(join);
      배열명.join('구분자');
      원래 배열 frontLangs 의 값은 join 함수를 사용해도 변하지 않음

reverse

        원소의 정렬를 역순 변경(원본 수정!)
        배열명.reverse();

sort

        원소들을 알파벳순, 오름차순(작은 수->큰 수)으로 정렬 (원본 수정!)
        sort의 매개변수 -을 입력해 역순, 알파뱃순 반대, 숫자는 내림차순으로 정렬 (원본 수정!)
        return 값을 0이나 1로 변경하면 원소의 배열을 변경하지 않게 됨
        배열명.sort();

push

기존 배열에 마지막에 값을 추가

pop

기존 배열에 마지막 값을 제거

unshift

배열의 처음에다 값을 추가

shift

배열의 처음에다 추가 된 값 제거

slice

배열에서 원하는 부분 복사.원본보존

splice

배열에서 원하는 부분 잘라내기. 원본바뀜

concat

       새로운 배열 = 배열.concat(배열)
       기존 배열 원본유지

filter

        조건에 부합하는 원소들을 추출하여 새로운 배열을 생성
