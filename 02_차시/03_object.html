Object

     객체는 변수와 함수가 모여 만든 하나의 꾸러미
     함수 보다 좀 더 복잡하고 정교한 데이터들을 하나의 객체에 담아 활용할 수 있다.
     객체(object) = 변수(property) + 함수(method)

문법

  • var(let/const) 객체명 = new object()
  • var 객체명 = {} (아무값 정해져있지 않은 빈 객체)
  • 객체명, 프로퍼티명으로 객체 내 값 출력
  • 객체명, 프로터피명()로 객체 내 함수 실행 document.write(s1.math);
    s1안에 math라는 변수의 값을 출력해
    document.write(s2.total());
    s2안에 total이라는 객체 안에 있는 함수를 실행해
