이벤트

document.getElementById("target").addEventListener('click,function(){ //이벤트 발생시 수행할 코드 });

  1. 이벤트를 적용할 대상을 지정한다
  2. 이벤트 종류를 입력한다
  3. 이벤트가 발생했을 때 수행할 코드를 기술한다
  • this는 이벤트가 발생한 객체를 참조할 수 있다.
  • click: 클릭할 때
  • mouseover : 마우스가 특정 객체 위로 올려졌을 때 조금이라도 마우스가 움직이면 반복됨
  • mouseout : 마우스가 특정 객체 밖으로 나갔을 때 조금이라도 마우스가 움직이면 반복됨
  • mouseenter : 마우스가 특정 객체 안에 들어왔을 때
  • mouseleave : 마우스가 특정 객체에서 떠났을 때
  • mousedown : 마우스를 클릭했을 때
  • mouseup : 마우스에서 손을 뗐을 때
  • scroll : 문서에서 스크롤을 할 때
  • change : input 요소의 변동이 있을 때
  • focus : 링크, input등의 요소에 포커스를 얻었을 때
  • keydown : 키를 눌렀을 때
  • keyup : 키에서 손을 뗐을 때
