Timer

setTimeout

  • 할일 = 함수 = function(){}
  • 시간은 millisecond 천분의 1초, 1000=1초
  • setTimeout(할일,시간)

clearTimeout

  • setTimeout함수로 지정한 할 일을 멈춰야 할 때 사용
  • 멈추려는 대상의 이름을 매개변수로 지정한다
  • clearTimeout(대상);
  • 이때 setTimeout함수는 이름이 없는 함수이기 때문에, 변수에 할당해 이름을 지정한다

setInterval

  • 일정 시간마다 반복적으로 지정한 코드를 실행할 수 있다.
  • 웹페이지 슬라이드 넘어갈 때 활용

clearInterval

  • 멈추려는 대상의 이름을 매개변수로 지정한다
  • clearInterval(대상);
  • 이때 setInterval 함수는 이름이 없는 함수이기 때문에, 변수에 할당해 이름을 지정한다
